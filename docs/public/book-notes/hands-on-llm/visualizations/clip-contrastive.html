对比学习损失（InfoNCE）

对于 batch 中的 N=4 对图文：

构造相似度矩阵 S[i][j] = cos(image_i, text_j)

损失函数（简化版）：
L = -Σᵢ log[ exp(S[i][i]/τ) / Σⱼ exp(S[i][j]/τ) ]

目标：让对角线 S[i][i] 最大，
非对角线 S[i][j], j≠i 最小。

τ (温度)：控制 softmax 分布的 sharpness，
CLIP 用可学习的 τ ≈ 0.07。
为什么这样有效？
单看一条 (image, text) 对，模型无法知道"什么是好"。 但放进一个 batch 和其他 N-1 个负样本一起做 softmax 归一化后， 模型必须学会"为什么 image_A 应该匹配 text_A 而不是 text_B/C/D"， 才能把对角线 push up、其他位置 push down。

4 亿对图文 + 这个 loss 就是 CLIP 的全部秘密。

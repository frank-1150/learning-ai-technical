# 技术书籍阅读

这里收录的是我在阅读技术书籍过程中整理的深度笔记。每一本书会被拆解为若干篇独立的技术文章，配合交互式可视化，力求将书中的核心思想用更直观的方式呈现出来。

与单纯的读书摘要不同，这些笔记更侧重于 **提炼可复用的设计模式和工程原则** ——那些读完一遍可能记不住、但在实际系统设计中反复出现的东西。
